# 英语句子分析与标记任务说明

## 任务目标
为英语学习材料中的英文句子自动标记谓语动词和固定搭配，帮助学习者更好地理解句子结构，提高英语学习效率。

## 任务步骤
1. **分析项目结构**：首先了解现有的HTML文件结构和CSS样式定义
2. **创建样式文件**：创建sentence-analysis.css文件，定义谓语和固定搭配的样式
3. **引入样式表**：在目标HTML文件的head部分引入新创建的样式表
4. **标记英文句子**：遍历所有translation-block组件，为英文原句中的谓语动词和固定搭配添加标记
5. **验证效果**：确保所有标记正确应用，页面显示正常

## 样式要求
- 谓语动词：蓝色（#3b82f6）、字重500
- 固定搭配：紫色（#8b5cf6）、字重500、点状下划线
- 鼠标悬停效果：轻微放大和过渡动画

## 标记规范
- 使用`<span class='english-predicate'>`标签标记谓语动词
- 使用`<span class='english-collocation'>`标签标记固定搭配
- 标记应精确，不遗漏任何主要动词或动词短语
- 保留原始句子的其他内容不变

## 文件路径约定
- 样式文件：components/sentence-analysis.css
- 目标文件：lessons/001.html, lessons/002.html等
- 组件样式：components/translation-component.css

## 注意事项
- 确保所有HTML标签正确闭合
- 保持原始文件的其他结构不变
- 注意句子中的特殊字符和标点符号
- 对于复杂句子，确保准确识别主要谓语部分
- 考虑句子的语法结构和语义关系

## 示例
### 原句：
The other technologies make css and javascript easier to work with

### 标记后：
The other technologies <span class='english-predicate'>make</span> css and javascript easier to work with

### 复杂句子示例：
Let me <span class='english-predicate'>know</span> in the comments how many of these technologies you already <span class='english-predicate'>knew</span>, and if anything <span class='english-predicate'>was</span> new to you

### 固定搭配示例：
We're <span class='english-collocation'>going to look at</span> how the frontend <span class='english-collocation'>communicates with</span> the backend

## 使用说明
1. 将此prompt提供给AI助手
2. 指定需要分析的HTML文件路径
3. AI助手会自动分析并标记所有英文句子
4. 完成后可以直接在浏览器中打开文件查看效果
